# CDU 07. Visualizar Post

- **Ator principal**: Cliente e Visitante.

- **Atores secundários**: N/A.
- **Resumo**: O Cliente/Visitante entra na página de investimentos e visualiza os posts publicados.
- **Pré-condição**: Entrar na página de investimentos.
- **Pós-Condição**: N/A.

## Fluxo Principal

| Ações do ator | Ações do sistema |
| :-----------------: | :-----------------: |
| 1 - O Cliente/Visitante entra na página investimentos. | |
| | 2 - Retorna na tela os posts publicados. |
