# CDU 25. Visualizar Dashboard 

- **Ator principal**: Usuário.
- **Atores secundários**: Não existe.	 
- **Resumo**: Através da pagina dashboard o usuário visualiza um resumo geral sobre suas transações, gastos, receitas e depesas.
- **Pré-condição**: Usuário autenticado
- **Pós-Condição**: O dashboard exibe os dados financeiros do usuário 

## Fluxo Principal

| Ações do ator | Ações do sistema |
|---------------|------------------|
| 1. O usuário acessa a página do dashboard. | |
| | 2. O sistema recupera as transações financeiras associadas ao usuário |
| | 3. O sistema processa os dados (total de gastos, receitas e despesas). |
| | 4. O sistema exibe o dashboard com o resumo financeiro. |
|  | |
