# CDU 31. Alterar Foto da Família

- **Ator principal**: Cliente.
- **Atores secundários**: N/A.
- **Resumo**: O Cliente pode alterar a foto de identificação da família realizando o upload de uma nova imagem.
- **Pré-condição**: O Cliente deve estar autenticado e possuir papel de administrador para editar a família.
- **Pós-Condição**: A nova foto da família é salva e passa a ser exibida na plataforma.

## Fluxo Principal

| Ações do ator | Ações do sistema |
| :-----------------: | :-----------------: |
| 1 - O Cliente acessa as configurações da família. | |
| 2 - O Cliente seleciona a opção **"Alterar Foto"**. | |
| | 3 - O sistema abre a interface para seleção da imagem. |
| 4 - O Cliente seleciona a nova foto e confirma o envio. | |
| | 5 - O sistema realiza o upload e atualiza a foto da família. |
| | 6 - O sistema exibe uma mensagem de sucesso. |
