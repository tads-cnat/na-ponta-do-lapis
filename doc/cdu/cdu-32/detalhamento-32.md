# CDU 32. Alterar Nome da Família

- **Ator principal**: Cliente.
- **Atores secundários**: N/A.
- **Resumo**: O Cliente pode alterar o nome da família por meio das configurações.
- **Pré-condição**: O Cliente deve estar autenticado e possuir papel de administrador para editar a família.
- **Pós-Condição**: O nome da família é atualizado no sistema.

## Fluxo Principal

| Ações do ator | Ações do sistema |
| :-----------------: | :-----------------: |
| 1 - O Cliente acessa as configurações da família. | |
| 2 - O Cliente seleciona a opção **"Alterar Nome"**. | |
| | 3 - O sistema exibe o formulário para edição do nome da família. |
| 4 - O Cliente informa o novo nome e confirma a operação. | |
| | 5 - O sistema valida, atualiza o nome da família no banco de dados e exibe uma mensagem de sucesso. |
