# Na Ponta do Lápis

<img src="logo.png" width="200" height="200" />

Nosso sistema web cria um ambiente digital acolhedor e interativo voltado para jovens que estão iniciando a vida adulta. A plataforma oferece ferramentas inteligentes de controle financeiro, planejamento de gastos e educação financeira personalizada. Com uma experiência prática e intuitiva, o sistema ajuda o usuário a desenvolver hábitos saudáveis de gestão do dinheiro, compreender o impacto de suas decisões financeiras e construir uma base sólida para o crescimento pessoal e profissional.
