PRÉ-REQUISITOS:
  • Django server rodando: python manage.py runserver

  • Estar logado em uma conta

TESTE 1: Validação de Nome (Vazio)
═══════════════════════════════════════════════════════════════════════════════

PASSOS:
  1. Navegue para http://127.0.0.1:8000/contas/
  2. Clique em "Adicionar Conta" (ou "Criar Primeira Conta")
  3. Modal abre com campos vazios
  4. Deixe o campo "Nome da Conta" vazio
  5. Clique no botão "Salvar"

RESULTADO ESPERADO:
  ✅ Dentro do modal, em vermelho:
     "Erros encontrados:
      • Nome da conta é obrigatório"
  ✅ Modal não fecha
  ✅ Formulário não é enviado

TESTE 8: Editar Conta
═══════════════════════════════════════════════════════════════════════════════

PASSOS:
  1. Na página de contas, clique no botão "Editar" de uma conta
  2. Modal de edição abre
  3. Tente os mesmos testes de validação (nome vazio, tipo etc)
  4. Preencha corretamente com novos dados
  5. Clique "Salvar Alterações"

RESULTADO ESPERADO:
  ✅ Mesma validação que "Adicionar Conta"
  ✅ Modal com exibição de erros
  ✅ Mensagem de sucesso em verde quando salvo

TESTE 9: Múltiplas Contas
═══════════════════════════════════════════════════════════════════════════════

PASSOS:

  2. Navegue com os botões ← e →
  3. Clique nos dots para ir a uma conta específica
  4. Verifique que cada conta mostra os dados corretos

RESULTADO ESPERADO:
  ✅ Carousel funciona com qualquer número de contas
  ✅ Navegação fluida
  ✅ Cada botão CRUD funciona para a conta correta
