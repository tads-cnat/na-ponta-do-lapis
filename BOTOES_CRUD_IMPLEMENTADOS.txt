🎯 FLUXO PASSO-A-PASSO
═══════════════════════════════════════════════════════════════════════════════

FLUXO 1: Editar Conta
──────────────────────
1. Usuário navega até uma conta (carrossel move)
2. Conta no centro é "selecionada" (offset = 0)
3. Clica em "Editar Conta"
4. carousel.js detecta clique, pega getCurrentAccountId()
5. Chama modalManager.openEditAccountModal(id)
6. Modal abre e loadAccountData(id) é acionado
7. Fetch para /contas/api/obter/<id>/
8. Servidor valida autenticação e propriedade
9. Retorna JSON: {id, nome, saldo, tipo, sucesso: true}
10. JavaScript preenche todos os inputs
11. Tipo é selecionado automaticamente (botão fica destacado)
12. Usuário edita os dados que desejar
13. Clica "Salvar Alterações"
14. Validação frontend verifica campos
15. Form submete via POST para /contas/editar/<id>/
16. Backend valida e atualiza conta
17. Redirect para /contas/ com mensagem de sucesso
18. Página recarrega
19. Carousel mostra conta atualizada
20. Mensagem "Conta 'X' atualizada com sucesso!" aparece por 5s

FLUXO 2: Excluir Conta
──────────────────────
1. Usuário navega até uma conta
2. Clica em "Excluir Conta"
3. carousel.js pega getCurrentAccountId()
4. Chama modalManager.openDeleteAccountModal(id)
5. Modal de confirmação abre
6. Modal mostra aviso: "Ação irreversível"
7. Usuário clica "Sim, desejo excluí-la"
8. confirmDeleteAccount() cria formulário POST
9. Adiciona CSRF token ao formulário
10. Submete para /contas/excluir/<id>/
11. Backend valida propriedade
12. Deleta conta do banco de dados
13. Redirect com mensagem de sucesso
14. Página recarrega
15. Carousel atualiza (conta desaparece)
16. Se era a última conta, mostra botão "Criar Primeira Conta"
17. Mensagem "Conta 'X' excluída com sucesso!" aparece por 5s

FLUXO 3: Botões Desabilitados
──────────────────────────────
1. Usuário sem contas criadas
2. Página carrega com tem_contas = False
3. Script index.html detecta: !temContas
4. Desabilita botões: viewAccountBtn, editAccountBtn, deleteAccountBtn
5. Adiciona classes: btn-disabled, opacity-50, border-gray-300, text-gray-400
6. Botões ficam esmaecidos e não respondem a cliques
7. Apenas botão "Criar Primeira Conta" está ativo
8. Usuário cria primeira conta
9. Página recarrega com tem_contas = True
10. carousel.js ativa todos os botões CRUD
11. Botões voltam à cor normal e respondem a cliques
